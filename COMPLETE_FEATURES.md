## How to Maintain & Customize

### Updating Navigation
Edit `components/Header.tsx`:
```typescript
const navItems = [
  { label: 'Label', id: 'section-id' },
  // ...
];
```

### Adding New Programs
1. Update `components/Programs.tsx` with new program
2. Add slug mapping
3. Create new program page in `app/programs/[slug]/page.tsx`

### Updating Colors
Edit `app/globals.css`:
```css
:root {
  --primary: #06b6d4;    /* Teal */
  --secondary: #ff9500;  /* Orange */
  /* ... */
}
```

### Customizing Content
Each component has clearly separated:
- Data arrays (programs, trainers, features)
- JSX structure
- Styling classes
