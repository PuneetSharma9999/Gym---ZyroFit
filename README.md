## 🧠 Overview  

**ZyroFit Gym** is a modern, fully responsive gym website built using **Next.js and Tailwind CSS**.  

It is designed as a **customizable template** for gyms, fitness centers, or personal trainers to quickly launch a professional website.  

The project focuses on **clean UI, modular components, and easy customization**.

---

## 🔥 Features  

### 🎯 Clean Landing Page  
- Modern hero section with CTA  
- Structured sections  

### 💪 Training Programs  
- Easily customizable program list  
- Duration & descriptions  

### 👥 Trainer Profiles  
- Trainer profiles with specialization & experience  

### 💳 Membership Plans  
- Multiple pricing tiers  
- Highlighted featured plan  

### ⭐ Testimonials  
- Client reviews & ratings  

### 📞 Contact Section  
- Contact form UI  
- Address, phone, email  

### 🌙 Dark Mode  
- Light/Dark theme toggle  

### ⚡ Performance  
- Optimized with Next.js  

### 📱 Fully Responsive  
- Works on all devices  

---

## 🏗️ System Flow  
Landing Page → Explore Sections → Programs → Trainers → Pricing → Contact
